# Math & Philosophy Password Generator 🔑

Projekt w Pythonie generujący losowe hasła inspirowane filozofami i elementami matematyki.

## Funkcje

- Losowy wybór filozofa (imię, miasto, szkoła filozoficzna, rok urodzenia).
- Mieszanie wielkich i małych liter w tekście.
- Dodawanie symboli specjalnych (`! @ # $ % ^ & * ? ;`).
- Wykorzystanie operacji matematycznych (`sqrt`, `sin`, `cos`, liczby π i e) do generowania losowych liczb.
- Zapisywanie haseł do pliku `passwords.txt`.
